# Better Bash for Claude Code

## The problem?

As an AI agent, Claude Code runs commands to take actions on a user's behalf to interact with the filesystem and execute programs, but users of Claude Code who are new to terminals trouble following along. This discourages command line skill acquisition, which in my experience, is fundamental to fluency in computer literacy. A person who has mastery of data structures and algorithms but cannot navigate a terminal is highly intelligent, and yet, highly helpless.

## The solution?

Using colors to create visual contrast, this Claude Code plugin is an educative tool that highlights commands. This draws from my childhood experiences highlighting parts of speech while learning various human languages.

## Quickstart guide

Windows users: If emoji are not rendering properly in Git Bash, please download the latest version of Git Bash (emoji support was added in v2.51.0)
